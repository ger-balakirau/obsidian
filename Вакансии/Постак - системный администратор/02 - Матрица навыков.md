# Матрица навыков

Матрица помогает видеть, что уже закрыто в базе знаний, а где нужны новые заметки и практика.

| Навык            |       Важность | Что надо уметь                               | Статус         |
| ---------------- | -------------: | -------------------------------------------- | -------------- |
| TCP/IP           |        высокая | IP, маска, шлюз, подсеть, маршрутизация      | частично есть  |
| DHCP             |        высокая | выдача адресов, scope, lease, диагностика    | нужно добавить |
| DNS              |        высокая | A/CNAME, resolver, доменный DNS, диагностика | нужно добавить |
| Active Directory |        высокая | пользователи, группы, OU, join domain        | нужно добавить |
| GPO              |        высокая | применение политик, gpupdate, gpresult       | нужно добавить |
| WSUS             |        средняя | обновления Windows, группы компьютеров       | нужно добавить |
| Windows Server   |        высокая | роли, службы, event logs, shares             | нужно добавить |
| Windows клиент   |        высокая | установка, драйверы, сетевые настройки       | нужно добавить |
| RAID             |        высокая | уровни 0/1/10/5/6, degraded, rebuild         | нужно добавить |
| VMware ESXi      |        средняя | VM, datastore, snapshot, network             | нужно добавить |
| Debian Linux     |        средняя | пользователи, SSH, systemd, сеть             | частично есть  |
| OpenVPN          |        средняя | server/client, routes, firewall              | нужно добавить |
| Veeam            |        средняя | jobs, repository, restore, alerts            | нужно добавить |
| Zabbix           |        средняя | hosts, items, triggers, alerts               | нужно добавить |
| 1С               |        средняя | пользователи, backup, базовая архитектура    | нужно добавить |
| СКС              |        средняя | кабель, патч-панель, прозвонка               | нужно добавить |
| IP-камеры        |        средняя | IP, PoE, RTSP, NVR, доступность              | нужно добавить |
| СКУД Rusguard    | низкая/средняя | базовые принципы, контроллеры, доступ        | нужно добавить |
| Helpdesk         |        высокая | коммуникация, приоритизация, заявки          | нужно добавить |
