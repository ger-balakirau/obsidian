# Разбор вакансии

## Роль

Это позиция системного администратора широкого профиля. Нужно поддерживать офисную инфраструктуру, пользователей, Windows Server, сети, физические серверы, резервное копирование, мониторинг и часть специфичных систем: 1С, СКУД, видеонаблюдение.

## Обязанности из вакансии

- Поддержка пользователей в офисе и удалённо.
- Парк компьютеров 50+.
- Организация рабочих мест: подбор, закупка, сборка, настройка.
- Установка Windows и прикладного ПО на клиентские ПК.
- Работа с физическими серверами Supermicro.
- Обслуживание оргтехники.
- Развёртывание и обслуживание Windows Server.
- Базовая прокладка и диагностика СКС.
- Настройка маршрутизаторов, точек доступа, IP-камер.
- Взаимодействие с операторами связи.
- Резервное копирование данных.

## Требования

- Техническое образование или обучение на старших курсах.
- Опыт от 1 года.
- Английский на уровне чтения документации.
- Коммуникабельность, вежливость, стрессоустойчивость.
- Желание развиваться.

## Технические навыки

- Аппаратная часть ПК.
- RAID: 0, 1, 10, 5, 6.
- Windows XP/Vista/7/8/10/11 на уровне администрирования.
- TCP/IP, IP-адрес, маска, шлюз, подсеть, маршрутизация.
- ICMP, POP/IMAP/SMTP, SSH, FTP, HTTP/HTTPS, DHCP, DNS, VPN, NAT.
- VMware ESXi / vSphere.
- Rusguard.
- Видеонаблюдение 100+ камер.
- Debian Linux, OpenVPN.
- Windows Server 2012 R2 / 2019.
- Active Directory, DNS, WSUS, GPO.
- 1С Предприятие: разворачивание, backup, пользователи.
- Veeam.
- Zabbix.

## Риски на собеседовании

Могут спросить не “что такое AD”, а практические вещи:

- пользователь не может войти в домен;
- компьютер не получает IP;
- GPO не применяется;
- DNS работает странно;
- сервер не видит сеть;
- резервная копия не прошла;
- диск в RAID умер;
- камера не открывается;
- OpenVPN подключается, но нет доступа в LAN;
- служба Windows не стартует;
- 1С база не открывается у части пользователей.

## Главный вывод

Нужно готовиться как к роли администратора инфраструктуры: пользователи + Windows-домен + сеть + backup + мониторинг + базовый Linux.
