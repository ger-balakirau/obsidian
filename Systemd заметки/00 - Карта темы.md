# Systemd - карта темы

`systemd` управляет сервисами, автозапуском, логами, таймерами и состоянием системы. Для администрирования Linux это одна из ключевых тем.

## Что должно стать понятно

- Что такое unit.
- Чем отличаются `service`, `timer`, `target`.
- Как запускать, останавливать и перезапускать сервисы.
- Как включать автозапуск.
- Где лежат unit-файлы.
- Как читать логи через `journalctl`.
- Как диагностировать сервис, который не стартует.

## Минимум на память

```bash
systemctl status nginx
sudo systemctl start nginx
sudo systemctl restart nginx
sudo systemctl enable nginx
journalctl -u nginx -f
sudo systemctl daemon-reload
```
