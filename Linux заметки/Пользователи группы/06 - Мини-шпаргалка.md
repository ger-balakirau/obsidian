
# Мини-шпаргалка: пользователи, группы и права

## Проверка

```bash
whoami                   # текущий пользователь
id                       # uid, gid и группы
groups                   # группы текущего пользователя
ls -l                    # права файлов
ls -ld dir               # права самой директории
namei -l /path/to/file   # права всех директорий в пути
stat file                # подробности файла
```

## Обычные права

```bash
chmod 644 file           # обычный файл
chmod 600 secret         # приватный файл
chmod 755 dir            # обычная директория
chmod 700 private        # приватная директория
chmod +x script.sh       # сделать файл исполняемым
```

## Владелец и группа

```bash
chown user file          # изменить владельца
chown user:group file    # изменить владельца и группу
chown -R user:group dir  # рекурсивно для директории
chgrp group file         # изменить группу
```

## Особые права

```bash
chmod u+s file           # setuid
chmod g+s dir            # setgid для директории
chmod +t dir             # sticky bit
```

## Ссылки и inode

```bash
ls -li                   # inode + права
ln file hard             # hard link
ln -s file link          # symlink
readlink link            # цель symlink
```

## umask и ACL

```bash
umask                    # посмотреть umask
umask 022                # обычные новые права: file 644, dir 755
umask 077                # приватные новые права: file 600, dir 700

getfacl file             # посмотреть ACL
setfacl -m u:user:rw file
setfacl -m g:group:rw file
setfacl -d -m g:group:rwx dir
setfacl -b file          # убрать дополнительные ACL
```

## Самая частая проверка при `Permission denied`

```bash
whoami
id
ls -l file
ls -ld dir
namei -l /full/path/to/file
```
