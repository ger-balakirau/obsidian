Раздел про установку и базовую эксплуатацию Docker Engine на Linux-сервере.

## Что должно стать понятно

- Чем Docker Engine отличается от Docker Desktop.
- Как быстро поставить Docker через официальный convenience script.
- Как поставить Docker Engine через официальный apt-репозиторий.
- Зачем нужен пакет `docker-compose-plugin`.
- Почему после добавления пользователя в группу `docker` нужно перелогиниться.
- Как проверить `docker` и `docker compose`.
- Как создавать и удалять контейнеры, образы, volumes и networks.
- Какие `prune`-команды опасны и что именно они удаляют.

## Минимум на память

```bash
curl -fsSL https://get.docker.com -o get-docker.sh
sudo sh get-docker.sh
sudo usermod -aG docker $USER
newgrp docker
docker run hello-world
docker compose version
docker ps -a
docker images
docker volume ls
docker network ls
docker system df
```
