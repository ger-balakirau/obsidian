## Минимальная шпаргалка

Полная командная шпаргалка: [[19 - Команды SSH - общая шпаргалка]].

```bash
ssh user@host
ssh work
ssh -p 2222 user@host
ssh -i ~/.ssh/id_ed25519 user@host
ssh -v user@host
ssh-copy-id user@host
scp file.txt user@host:/tmp/
```

## Если что-то не работает

1. Запусти подключение с `-v`: `ssh -v user@host`.
2. Проверь имя пользователя, порт и ключ.
3. Проверь права на сервере: `~/.ssh` должен быть `700`, `authorized_keys` — `600`.
4. Посмотри [[15 - Диагностика SSH-подключения]] и [[17 - Частые ошибки SSH]].
