## Пользователь в SSH-команде

В SSH пользователь указывается перед адресом сервера:

```bash
ssh user@server
```

Пример:

```bash
ssh root@185.10.20.30
```

```bash
ssh deploy@example.com
```

Формат:

```text
user   — имя пользователя на сервере
server — IP-адрес или домен сервера
```

---

## Домашняя директория пользователя

У каждого пользователя на сервере есть домашняя директория.

Обычно:

```text
/home/user
```

Для пользователя `root`:

```text
/root
```

Примеры:

```text
/home/deploy
/home/admin
/root
```

---

## Что происходит после входа

После подключения по SSH пользователь обычно попадает в свою домашнюю директорию.

Пример:

```bash
ssh deploy@server.com
```

После входа текущая директория чаще всего будет:

```text
/home/deploy
```

---

## Символ `~`

Символ `~` означает домашнюю директорию текущего пользователя.

```bash
cd ~
```

То же самое:

```bash
cd /home/user
```

Примеры:

```bash
cd ~/.ssh
```

```bash
ls ~
```

---

## Проверка текущего пользователя и директории

Показать текущего пользователя:

```bash
whoami
```

Показать текущую директорию:

```bash
pwd
```

Пример:

```text
deploy
/home/deploy
```

# Главное запомнить

```bash
ssh user@server
```

`user` — это пользователь на сервере.

Домашняя директория обычного пользователя:

```text
/home/user
```

Домашняя директория root:

```text
/root
```

`~` — домашняя директория текущего пользователя.
