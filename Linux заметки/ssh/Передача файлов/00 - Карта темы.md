## Как выбрать инструмент

### `scp`

Используй, когда нужно быстро передать файл или папку.

```bash
scp file.txt work:/tmp/
```

### `rsync`

Используй, когда нужно синхронизировать директории, передавать только изменения или делать деплой.

```bash
rsync -av ./project/ work:/srv/project/
```

### `sftp`

Используй, когда хочешь интерактивно посмотреть директории на сервере и вручную выбрать файлы.

```bash
sftp work
```
