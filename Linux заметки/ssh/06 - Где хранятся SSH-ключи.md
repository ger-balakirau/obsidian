# Где хранятся SSH-ключи

## Основная директория

SSH-ключи обычно хранятся в директории:

```text
~/.ssh
```

Полный путь для обычного пользователя:

```text
/home/user/.ssh
```

Для пользователя `root`:

```text
/root/.ssh
```

---

## Приватный и публичный ключ

Обычно ключ состоит из двух файлов:

```text
id_ed25519
id_ed25519.pub
```

Где:

```text
id_ed25519      — приватный ключ
id_ed25519.pub  — публичный ключ
```

---

## Приватный ключ

Приватный ключ хранится на твоём компьютере.

Пример:

```text
~/.ssh/id_ed25519
```

Его нельзя передавать другим людям и выкладывать в публичные места.

---

## Публичный ключ

Публичный ключ можно добавлять на сервер.

Пример:

```text
~/.ssh/id_ed25519.pub
```

Именно его содержимое добавляют в файл:

```text
~/.ssh/authorized_keys
```

---

## Частые имена файлов ключей

```text
id_ed25519
id_ed25519.pub
id_rsa
id_rsa.pub
```

---

## Главное запомнить

SSH-ключи обычно лежат здесь:

```text
~/.ssh
```

Приватный ключ:

```text
id_ed25519
```

Публичный ключ:

```text
id_ed25519.pub
```

Приватный ключ никому не передавать.
