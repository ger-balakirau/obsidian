# Ansible - карта темы

Ansible - инструмент для управления серверами по SSH: установка пакетов, настройка конфигов, запуск сервисов, деплой и повторяемая автоматизация.

## Рекомендуемый порядок

1. [[01 - Быстрый запуск playbook]]
2. [[02 - Inventory и подключение по SSH]]
3. [[03 - Глубокий разбор playbook]]

## Что должно стать понятно

- Как запустить playbook.
- Как описать серверы в inventory.
- Как подключаться по SSH-ключу.
- Как подключаться по паролю.
- Как использовать `become` для sudo.
- Из чего состоит playbook: plays, tasks, modules, vars, handlers.
- Как запускать часть playbook через tags.
- Как проверять ошибки подключения.

## Минимум на память

```bash
ansible --version
ansible all -i inventory.ini -m ping
ansible-playbook -i inventory.ini site.yml
ansible-playbook -i inventory.ini site.yml --ask-pass
ansible-playbook -i inventory.ini site.yml --ask-become-pass
ansible-playbook -i inventory.ini site.yml --check
ansible-playbook -i inventory.ini site.yml --syntax-check
```
