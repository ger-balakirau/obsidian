## Что учить сначала

### 1. Командная строка

Цель: уверенно перемещаться по системе, искать файлы, читать логи и фильтровать вывод команд.

Минимум:

```bash
pwd
ls -la
cd
cp
mv
rm
find
grep
less
tail -f
```

### 2. Пользователи и права

Цель: понимать, почему файл открывается или не открывается, почему сервис не может прочитать конфиг, почему SSH ругается на права.

Минимум:

```bash
whoami
id
ls -l
chmod
chown
chgrp
namei -l
```

### 3. SSH

Цель: безопасно подключаться к серверу, работать с ключами и диагностировать ошибки входа.

Минимум:

```bash
ssh user@host
ssh -i ~/.ssh/key user@host
ssh -v user@host
ssh-copy-id user@host
```

### 4. Сервер

Цель: после установки Ubuntu привести сервер в базовое рабочее состояние: обновления, пользователь, SSH, firewall, время, пакеты, swap, логи.

### 5. Git

Цель: сохранять историю изменений, работать с ветками и синхронизировать проект с удалённым репозиторием.

Минимум:

```bash
git status
git add .
git commit -m "message"
git pull
git push
```

### 6. Troubleshooting

Цель: не угадывать, а проверять систему по слоям: сеть, ресурсы, процесс, сервис, логи, конфигурация.

## Правило для заметок

- Объяснение держим в тематической заметке.
- Команды держим в шпаргалке.
- Повторяем только короткий минимум, а за деталями ставим ссылку.
