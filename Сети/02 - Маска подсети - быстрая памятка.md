# Маска подсети - быстрая памятка

## 1. Через CIDR (/n)

- **n** — количество бит сети
- Всего адресов: `2^(32 - n)`
- Доступных: `-2` (сеть и broadcast)

**Примеры:**

- /24 → 256 (254 usable)
- /26 → 64 (62 usable)

## 2. Через маску (255.x.x.x)

| Значение | Биты |
| -------- | ---- |
| 255      | 8    |
| 254      | 7    |
| 252      | 6    |
| 248      | 5    |
| 240      | 4    |
| 224      | 3    |
| 192      | 2    |
| 128      | 1    |
| 0        | 0    |

👉 Складываешь биты → получаешь `/n`

**Пример:**  
255.255.255.192 → 24 + 2 = **/26**

---

## ⚡ Быстрый алгоритм

- Есть `/n` → считаешь адреса через формулу
- Есть `255.x.x.x` → считаешь биты → получаешь `/n`
