Частные сети (где используются)

- "10.0.0.0/8" → "255.0.0.0"
  → корпоративные сети, отели, кампусы, mesh/VLAN

- "172.16.0.0/12" → "255.240.0.0"
  → корпоративные сети, VPN, дата-центры

- "192.168.0.0/16" → "255.255.0.0"
  → домашние сети, малый бизнес

---

Типовые подсети

- "/24" → "255.255.255.0" → ~254 хоста
- "/16" → "255.255.0.0" → ~65k
- "/8" → "255.0.0.0" → ~16M

---

Часто используемые

- "192.168.0.0/24" → шлюз ".1"
- "192.168.1.0/24" → шлюз ".1"
- "10.x.x.0/24" → шлюз ".1"

---

Специальные

- "169.254.0.0/16" → нет DHCP
- "127.0.0.1" → localhost

---

Минимум для статики

- IP: "x.x.x.x"
- Mask: "/24"
- GW: ".1"
- DNS: "8.8.8.8"
