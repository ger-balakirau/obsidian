## Минимум на память

```bash
ip a
ip route
ping 8.8.8.8
ping ya.ru
nslookup ya.ru
traceroute ya.ru
ss -tulnp
nc -vz host 443
```

Windows:

```cmd
ipconfig /all
ping 8.8.8.8
nslookup ya.ru
tracert ya.ru
```
